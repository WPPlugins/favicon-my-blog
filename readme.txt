== Description ==

This plugin allows you to display a favicon on your blog - With maximum compatibility!

Simply enter the URLs of the favicon and your blog will use this as the favicon - Less than a minute setup time!

== Installation ==

1. Upload `/favicon-my-blog.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the Favicon My Blog Settings in your Administration Panel
4. Done!

== Frequently Asked Questions ==

= Does this work exactly as it should? =

Just enter the details the plugin asks for with the correct URLs and it should all work.
